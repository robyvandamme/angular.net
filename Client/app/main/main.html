Angular.NET

AngularJS meets ASP.NET: the Web API way.

Code on GitHub

Angular.NET is a proof of concept Single Page Application built using AngularJS and ASP.NET Web API 2. The idea behind it is to explore the capabilities and limitations of both frameworks and if possible develop a kind of blueprint/boilerplate for future use. Both Client and API are being continuously deployed to Azure Web Sites.
